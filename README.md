# MajorProjectPhase1

This repository contains the complete implementation code for Major Project Phase 1. Follow the instructions below to set up the environment and execute the project.

## Project Structure

```
MajorProjectPhase1/
│
├── main.ipynb
├── requirements.txt
├── README.md
├── runs/
```

## 1. Create Virtual Environment

Run the following command inside the project directory:

```
python -m venv venv
```

### Linux / macOS:

## 3. Install Required Libraries

After activating the virtual environment, install all project dependencies:

```
pip install -r requirements.txt
```

## 4. Running the Project

Open and run the main notebook:

```
main.ipynb
```

Execute the cells sequentially to reproduce the results.

## Notes

1. Use Python version 3.9 or higher.
2. If any package installation fails, update pip:
   ```
   pip install --upgrade pip
   ```
